# 📊 Data Analysis Project (Excel)

## 📌 Overview

This project is based on analyzing multiple datasets (Student, Sales, and Employee data) using Excel formulas. The data is taken from a dataset, not from a database.

---

## 📂 Datasets Used

1. Student Data (Marks & Grades)
2. Sales Data (Product & Amount)
3. Employee Data (Salary & Joining Date)

---

## 🔧 Work Done

### 1. Student Data Analysis

* Calculated Average marks using:
  =AVERAGE(C2:E2)
* Classified Grades using IF formula
* Used COUNTIFS to count students scoring above 60
* Used FILTER to extract students with marks > 80
* Identified students with English marks < 50

---

### 2. Sales Data Analysis

* Used VLOOKUP to fetch product price
* Used INDEX + MATCH for dynamic lookup
* Applied IFERROR to handle missing values

---

### 3. Employee Data Analysis

* Used XLOOKUP to fetch salaries
* Formatted Joining Date using TEXT
* Extracted Year and Month using YEAR and TEXT
* Calculated Experience using DATEDIF
* Categorized employees using IF
* Used COUNTIFS for year-wise joining analysis

---

## 📊 Key Formulas Used

* IF
* IFERROR
* VLOOKUP
* INDEX + MATCH
* XLOOKUP
* COUNTIF / COUNTIFS
* FILTER
* TEXT
* YEAR
* DATEDIF

---

## ✅ Conclusion

The project demonstrates how Excel formulas can be used to clean, analyze, and extract insights from datasets without using any database.
